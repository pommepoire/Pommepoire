# Pommepoire ✈️

Application de voyage partagé — PWA React + Firebase.

## Déploiement sur Vercel

1. Uploadez ce dossier sur GitHub (nouveau repo public ou privé)
2. Connectez-vous sur vercel.com avec votre compte GitHub
3. "New Project" → importez le repo → "Deploy"
4. C'est en ligne !

## Installer sur iPhone

1. Ouvrez l'URL Vercel dans **Safari**
2. Bouton Partager → "Sur l'écran d'accueil"
3. L'app s'installe comme une vraie application

## Stack technique

- React 18
- Firebase (Auth + Firestore)
- PWA (installable sur iPhone/Android)
- Déployé sur Vercel
